# ccalc :smoking:
Terminal Calculator Program written in C. Compile with 'make'.

## Features
Can be run as a continuous program or can be invoked with an expression to calculate.
Run ```./ccalc ?``` to view help.

## Supported Operations
* Addition
* Subtraction
* Multiplication
* Division
* Modulus
* Exponent
* Square Root
* Base10 to Binary
* Binary to Base10
